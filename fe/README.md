# HomeKitchenAI Frontend

This is a Next.js frontend for HomeKitchenAI, built with TypeScript, Tailwind CSS, and Sass support.

## Features
- Modular pages, components, and styles (Sass)
- Connects to backend API for recipe generation
- Modern UI with Tailwind and custom SCSS

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure
- `src/pages`: Page routes
- `src/components`: Reusable UI components
- `src/styles`: Global and modular SCSS styles

## API Integration
- Connects to backend at `/api/recipe` for recipe generation

## Customization
- Add your own components and styles in `src/components` and `src/styles`
